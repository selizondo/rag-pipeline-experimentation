# Setup and Usage

## Prerequisites

- Python 3.12+
- `uv` (recommended): `curl -LsSf https://astral.sh/uv/install.sh | sh`
- OpenAI-compatible API key (only required for answer generation and judging; all retrieval evaluation runs without it)

## Install

```bash
uv sync
# or: pip install -e .
```

## Configure

```bash
cp .env.example .env
# Edit .env: set LLM_API_KEY (llm_utils naming, not OPENAI_API_KEY)
```

```dotenv
LLM_API_KEY=sk-...
LLM_MODEL=gpt-4o-mini
LLM_BASE_URL=https://api.openai.com/v1
LLM_JUDGE_MODEL=gpt-4o-mini
```

## Quick Start (pre-downloaded corpus)

```bash
# Verify setup — 61 offline tests, no API calls
uv run pytest tests/ -q

# Run evaluation grid (pre-built indices and qrels included)
python scripts/evaluate.py data/papers/ data/qrels_filtered.json \
    --config config/experiments/baseline.yaml \
    -o experiments/results/ --force

# Generate answers, judge, visualize (requires LLM_API_KEY)
python scripts/run_full_pipeline.py --n-queries 10
```

## Running from Scratch

```bash
# Step 1: Download 100 arXiv papers
python scripts/download_dataset.py --limit 100

# Step 2: Build FAISS indices (20-120 min on CPU, subprocess-isolated per model)
python scripts/ingest_grid.py data/papers/ \
    --config config/experiments/baseline.yaml \
    --index-dir data/indices

# Step 3: Evaluate all 18 configs
python scripts/evaluate.py data/papers/ data/qrels_filtered.json \
    --config config/experiments/baseline.yaml \
    -o experiments/results/ --force
```

## Interactive Interfaces

```bash
# Streamlit web UI (sidebar: index, model, retrieval method, top-K)
streamlit run app.py

# CLI REPL
python scripts/serve.py -i data/indices/fixed_512_ol64__minilm-l6__hybrid_a0.6
```

| `config/experiments/smoke.yaml` | 1 | 1 chunker x 1 embedder x 1 retriever, 3 queries |
| `config/experiments/baseline.yaml` | 18 | 3 chunkers x 2 embedders x 3 retrievers, 50 queries per cell |

## Code Layout

```
rag-pipeline-experimentation/
├── app.py                          # Streamlit web UI
├── config/experiments/             # YAML experiment grid definitions
├── scripts/
│   ├── download_dataset.py         # Fetch arXiv PDFs + build qrels_filtered.json
│   ├── ingest_grid.py              # Pre-build FAISS indices (subprocess-isolated)
│   ├── precompute_queries.py       # Embed queries to cache before FAISS loads
│   ├── evaluate.py                 # Run experiment grid, save results
│   ├── serve.py                    # Interactive CLI REPL
│   └── run_full_pipeline.py        # Best result → generate → judge → visualize
├── src/
│   ├── pipeline.py                 # RAGPipeline: ingest() + query() + load()
│   ├── embedders.py                # SentenceTransformersEmbedder (lazy-load + disk cache)
│   ├── evaluator.py                # qrels I/O, per-query scoring, ExperimentResult
│   ├── experiment.py               # build_chunker/embedder/pipeline, run_grid()
│   ├── generator.py                # LLM answer generation with citation extraction
│   ├── judge.py                    # LLM-as-Judge (4 dimensions, 1-5 scale)
│   └── chunkers_ext.py             # RecursiveChunker, SlidingWindowChunker
├── data/
│   ├── papers/                     # 100 downloaded arXiv PDFs
│   ├── qrels_filtered.json         # 281 queries mapped to 100 papers
│   ├── query_cache/                # Pre-computed query embeddings (per model)
│   └── indices/                    # FAISS indices per (chunk_config x embed_model)
├── experiments/
│   ├── results/                    # JSON per completed experiment cell
│   └── iteration_log.jsonl         # Append-only run history
└── visualizations/                 # PNG charts (regenerated by run_full_pipeline.py)
```

## Common Issues

**`LLM_API_KEY is not set`:** Use `LLM_API_KEY`, not `OPENAI_API_KEY`. Copy `.env.example`.

**`ModuleNotFoundError: no module named 'rag_common'`:** Run `uv sync`. The shared library installs as an editable package from the git dependency.

**`No indices loaded` in Streamlit:** Run `scripts/ingest_grid.py` to build at least one index first.

**Slow first run:** SentenceTransformers downloads model weights on first use (~90MB MiniLM, ~420MB MPNet). Subsequent runs use cache.

**Charts not generating on headless servers:** Set `MPLBACKEND=Agg`.
